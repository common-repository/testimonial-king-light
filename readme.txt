== Description ==

Testimonials Live [Demo](http://tk.ineeditright.com).Testimonial King Light is an easy-to-use best testimonial slider plugin that allows users  to embed testimonials into a Page or Post using the shortcode with visual composer and gravatar support. Testimonial King also allows you to insert a list of all Testimonials or output a Random Testimonials.  Testimonials King allows you to include an image with each testimonial - this is a great feature for adding a photo of the testimonials author or its gravatar.Testimonial King Light also provide a testimonial form to submit testimonials.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the contents of `/testimonial-king/` to the `/wp-content/plugins/` directory
2. Activate  Testimonial King through the 'Plugins' menu in WordPress
3. Visit [here](https://tk.ineeditright.com) for information on how to configure and use the plugin.
